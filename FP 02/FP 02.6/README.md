# Exercício 6 - Conversão de Segundos

## Enunciado
Construa um programa para ler um intervalo de tempo em segundos, converter para horas, minutos e segundos e imprimir o resultado.  

## Objetivo
- Ler segundos.
- Converter para horas, minutos e segundos.
- Exibir formatado.

## Exemplo
Entrada: `3781`  
Saída: `01:03:01`
