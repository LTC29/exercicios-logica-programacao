# Exercício 3 - Perímetro do Triângulo

## Enunciado
Faça um programa que leia os três lados de um triângulo, calcule e imprima seu perímetro.  

## Objetivo
- Ler três lados.
- Calcular o perímetro (soma dos lados).
- Exibir o resultado.

## Exemplo
Entrada: `a=3, b=4, c=5`  
Saída: `Perímetro = 12`
